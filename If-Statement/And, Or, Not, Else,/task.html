And, Or, Not, Else

Often, we want to check if two conditions are true at the same time.
To do that, we use the operator and. It works like that: 


    if (oneCondition && anotherCondition) {
        //Only runs when both are true
    }

Similarly, there is or:


    if (oneCondition || anotherCondition) {
        //Only runs when at least one of them is true
    }

When you want to run something unless the condition is true, use not:


    if (!oneCondition) {
        //Only runs when one condition is not true
    }

Probably the most important feature is the keyword else:


    if (oneCondition) {
        //Run stuff
    } else if (anotherCondition) {
        //Only runs when oneCondition doesn't apply but anotherCondition does
    } else {
        //Only runs as a "fallback", when both conditions don't apply.
    }

Both the else if and else block are optional in an if-statement.
