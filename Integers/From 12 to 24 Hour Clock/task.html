12-Hour Clock, 24-Hour Clock

  • You get an Integer from 0 to 12 and a String that is either "AM" or "PM"
  • If the number is smaller than zero or bigger than 12, return -1 since the input is invalid
  • If the String is neither AM or PM, return -1
  • Return the time in 24-Hour Clock
When it's currently AM, the hour on a 24-Hour clock is the same as on a 12-Hour clock
When it's currently PM, the hour on a 24-Hour Clock is exactly 12 bigger than on a 12-Hour clock
