12-Hour Clock, 24-Hour Clock

  • Ask the user for the hour
  • If it's smaller than zero or bigger than 12, tell the user that their input is not valid and end the execution with return
  • Ask the user for AM or PM
  • If the user enters something that is not either AM or PM, tell the user that their input is not valid and ent the execution with return
  • Tell the user what time that is in 24-Hour-Clock
When it's currently AM, the hour on a 24-Hour clock is the same as on a 12-Hour clock
When it's currently PM, the hour on a 24-Hour Clock is exactly 12 bigger than on a 12-Hour clock
