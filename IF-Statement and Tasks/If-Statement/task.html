IF-Statement

We are going to take a look at if-statements. They work like that:

if (something) {
//Do stuff
}

The something is replaced with something that can be true or false. This is called a boolean. You already know some of these booleans. They occur when comparing two numbers. That's right, it's nothing else. So, an example of a program with an if-statement could be this:

val number = 3;
if (number < 6) {
print("The number is smaller than six!")
}

When the first condition fails, you may want to catch that. For that, use the else block.

val number = 3;
if (number < 6) {
print("The number is smaller than six!")
} else {
print("The number is not smaller than six!")
}

And for real complex things, use else if:

val number = 3;
if (number < 6) {
print("The number is smaller than six!")
} else if (number == 9) {
print("The number is equal to seven!")
} else {
print("The number is not smaller than six and not equal to 9.")
}
