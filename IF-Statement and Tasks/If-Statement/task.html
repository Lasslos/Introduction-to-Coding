IF-Statement

We are going to take a look at if-statements. They job is very important. You can add optional code, depending on what the situation is. 


    if (something) {
         //Do stuff
    }

The something is replaced with a boolean. It is a data type that has exactly two states: true or false.

You already know some of these booleans. They occur when comparing two numbers to each other. So, an example of a program with an if-statement could be this: 


    val number = 3
    if (number < 6) {
         print("The number is smaller than six!")
    }

You can enhance your if statement with the keywork else: 


    val number = 3
    if (number < 6) {
        println("The number is smaller than six!")
    } else {
        println("The number is not smaller than six")
    }

For even more specific cases, use the keyword else if: 


    val number = 3
    if (number < 6) {
        println("The number is smaller than six!")
    } else if (number == 42) {
        println("The number is equal to 42!")
    } else {
        println("The number is not smaller than six and not equal to 42!")
    }
