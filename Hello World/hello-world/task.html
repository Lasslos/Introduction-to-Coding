This is an introductory course to the Kotlin Programming Language, made by Laslo Hauschild. It is meant for absolute beginners.

Kotlin uses the Java Virtual Machine to run its code. That means, everything that works with Java works with Kotlin. Kotlin is the better Java. In 2016, Google announced to replace Java with Kotlin as the preferred language for Android App Development. And its usage go much further.

Your very first program will do the following: Output Hello World in the console. Use the hints below as instructions.

Write all of your code in the main function. This is the entry point of your program.
Use print("Your text here") to output something to the console.
