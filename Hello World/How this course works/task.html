How this course works

This course is made for absolute beginners. There will be different types of lessons.

  • Theory: Instead of reading the description, it should be taught by an experienced programmer. Always code along! That means that you always should try out, how something shown to you works. Instead of asking what if questions, try it yourself and then ask or make a comment!
  • Output: Coding exercise that is validated by output. It starts with the main function and uses print.
  • Edu: Coding exercise that is validated by tests. It starts with a custom function. These functions will get some information and return some information.
