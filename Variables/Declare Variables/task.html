Variables

Disclaimer: This lesson should be taught by an experienced programmer.

The second thing that computers are incredibly good in, is storing data. I am going to show you the most convenient way in kotlin.

Before we store data, we have to understand data types. You already know a few:

  • String [Text]
  • Integer, called Int [Whole numbers]
  • Double [Numbers with decimal places]

Here is how you store data:
var [name] = [some data]

That could look like that:
var data = "Hello World!"

You can also specify the type of data to be stored:
var data: String = "Hello World!"

If you want to change the value, simply reassign the variable:
data = "Something else"
That will delete the previous value in the variable, and put a new value into it.

There may be some cases where you want a variable to be unchangeable. It also helps to understand the code, so try to use this feature as often as possible.
val moreData = "This is unmodifiable, because it starts with the keyword val"

To actually access the data:
val data = "Hello World"
print(data)
