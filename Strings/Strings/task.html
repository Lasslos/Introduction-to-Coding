Theory: First Data Type: Text

Text is arguably one of the most important data types. Its name is String in almost all programming languages. Create a string by placing some text into quotation marks.

"This is a String!"

You can output this String by wrapping it into a print statement:
println("This is a String!")
We will not use that a lot.

Instead, we will use return "This is a String", since most of the output you make is checked by tests. Testing, however, is not easily possible when the output is displayed. That is why return, which passes the value to a method, in this case a testing method, will be used more
